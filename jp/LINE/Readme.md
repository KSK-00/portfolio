# LINE AI CONSULTATION AUTOMATION

> 日本の開発相談を、
> LINE上で受け付ける。

AIが相談内容を理解し、
相談の状態を保持し、
必要な情報を確認しながら、
ユーザーへ応答する。

**Built for real conversations,
not just demo messages.**
